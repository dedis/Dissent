Detailed Description

Represents a single instance of a cryptographically secure anonymous exchange.

The "V2" bulk protocol consists of a shuffle round which shares an anonymous DiffieHellman public component and public signing key. The cleartext in each transmission is the phase number, length of the next phase's message, a message, and a signature. The xor mask generation, distribution, and resolution are the same as "V1". See BulkRound comments for more information.
