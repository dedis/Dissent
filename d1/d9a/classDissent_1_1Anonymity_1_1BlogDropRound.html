const bool Dissent::Anonymity::BlogDropRound::VerifyAllProofs = false
static

If true, every server verifies all proofs before revealing the plaintext.

Otherwise, servers reveal the plaintext, and look at the proofs only if something went wrong.

NOTE: This option is only valid for Hashing and Pairing variants (NOT ElGamal)
